collective.nonprofitprogrammes Installation
----------------------------------------

To install collective.nonprofitprogrammes using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.nonprofitprogrammes`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.nonprofitprogrammes
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
